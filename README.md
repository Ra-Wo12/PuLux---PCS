# PuLux---PCS
Repositório projeto semestral de PCS - PuLux

 Implementações:
  - ESP32 (Placa mãe)
  - LDR (Sensor de Lux)
  - Botão (Acionar sistema)
  - Buzzer (Resposta tátil / sonora á resposta do LDR)
  - Bateria (No caso do sistema fisíco)
  - Site / App (Conexão ESP32 com celular para alterações em configurações)

Requesitos Funcionais
 - Informar usuário sobre iluminação ambiente
 - Notificar mudanças bruscas
 - Resposta Tátil (Som e/ou Vibração)
Requisitos Não-Funcionais
 - Conexão com celular - configuravél (tempo vibração e intensidade)
 - Segurança na conexão
 - Identificar iluminação ambiente precisamente (não dar resposta baseado em possíveis sombras, por exemplo)
 - Confortável no pulso
 - Acessível
 - Site no celular pensado para deficientes visuais
